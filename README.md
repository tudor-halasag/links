# Links — Tudor Halasag

Live: https://tudor-halasag.github.io/links

## Description
A single-page "link in bio" hub for Tudor Halasag — a simple, fast-loading
landing page that consolidates all key profiles and contact points into
one shareable URL.

## Purpose
- Central destination for social bio links (Instagram, etc.)
- Quick access to professional and personal profiles: GitHub, LinkedIn, Gmail
- Fitness/activity tracking links: Hevy, Strava
- Payments: Revolut
- Direct contact: phone, email

## Structure
- `index.html` — page markup
- `style.css` — styling, color palette, animations
- `tahlogo.svg` — header logo / footer refresh button
- `avatar.jpg` — profile photo (falls back to "TAH" monogram if missing)
- `apple-grey-icon.png` — favicon / browser tab icon

## Design
- Background `#2A324B`, header/footer `#767B91`, accents `#C7CCDB` / `#24262D`
- Animated bouncy entrance for link buttons
- Fonts: Oswald (display), Barlow (body)

## Hosting
Deployed via GitHub Pages from the `main` branch root.
